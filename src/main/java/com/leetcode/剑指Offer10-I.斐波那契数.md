```
class Solution {
    public int fib(int n) {
        int a = 0, b = 1 ,sum;
        for(int i = 0;i < n;i++){
            sum = (a + b) % 1000000007;
            a = b;
            b = sum;
        }
        return a;
    }
}
```
这道题其实最传统的做法就是使用递归，但是我们知道，若递归深度过大，就会导致栈溢出。

为了解决该问题，我们可以使用动态规划，将每次前两数之和存起来，便于下次直接使用，这样子，我们就把一个栈溢出的问题，变为了单纯的数学加法，大大减少了内存的压力。
